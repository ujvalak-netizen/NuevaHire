# 🚀 NuevaHire

**AI-Powered Hiring & Recruitment Platform**

NuevaHire is a modern recruitment platform that simplifies the hiring process using Artificial Intelligence. It helps recruiters source, evaluate, and manage candidates efficiently while providing applicants with a seamless job application experience.

![NuevaHire Logo](assets/logo.png)

---

## ✨ Features

- 🤖 AI-powered candidate screening
- 📄 Resume parsing and profile creation
- 🔍 Smart job matching
- 👥 Recruiter dashboard
- 📋 Job posting and management
- 📈 Candidate tracking system (ATS)
- 💬 Interview scheduling
- 📊 Hiring analytics and insights
- 📱 Responsive and modern UI
- 🔐 Secure authentication

---

## 🛠️ Tech Stack

### Frontend
- HTML5
- CSS3
- JavaScript
- Bootstrap / Tailwind CSS

### Backend
- Node.js
- Express.js

### Database
- MongoDB

### AI & APIs
- OpenAI API
- Resume Parsing API
- Email Notification Service

## 🚀 Getting Started

### Prerequisites

- Node.js
- npm
- MongoDB

### Installation

```bash
git clone https://github.com/yourusername/nuevahire.git

cd nuevahire

npm install

npm start
````

Open your browser and visit:

```
http://localhost:3000
```
